Detailed Description

Allows motors to be controlled via groups such as 'torso', 'head' and 'arm'.

This class facilitates grouping of the motors (and therefore joints) into subsystems, as in the YARP interface to the iCub. An instance of BodyPart therefore inherits a QVector of motors and provides a method setEncPos( const QVector<qreal>& x ), to set the encoder positions of the motors.
