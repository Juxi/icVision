Detailed Description

An interface that defines a handler for data arriving on YARP ports.

The handler is called by filters (RpcFilter and StreamFilter) and provides access to the data flowing through the filter. See VirtualSkin::StateObserver, VirtualSkin::CallObserver and VirtualSkin::ResponseObserver
