Detailed Description

Implements Joint::setM() for a prismatic (telescoping) joint.

The prismatic joint is not yet supported by the rest of the RobotModel library... How to define geometries such that collision detection can be done in a useful way is not straightforward. Since the iCub has no prismatic joints, this is on hold
