void CompositeObject::setT ( const QMatrix4x4 &  txfr  ) 

Sets the rotation/translation of the CompositeObject.

...by supplying the 4x4 transformation matrix directly. This is used to replace the transformation with a new one each time forward kinematics are computed. NOTE: There is no check that the supplied matrix represents a rigid-body transformation. non-rigid-body transformations have not been tested. Don't use them!
