Detailed Description

A kinematic model of a robot.

The robot model consists of a tree of KinTreeNode objects of type RevoluteJoint, PrismaticJoint and Link, as well as a list of Motor objects and a list of BodyPart objects. A Motor is bound to one or more joints and control the joints' position according to a linear mapping. A BodyPart is a groups of motors, the positions of which can be set all at once.
