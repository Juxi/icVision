Detailed Description

The QWidget responsible for rendering the scene with OpenGL.

This class, as well as Window have been almost entirely borrowed from the Qt HelloGL example. The only difference is that here arbitrary OpenGL display lists can be rendered. This is done in conjunction with the class RenderList.

The signals and slots between RenderList and this class work as follows: SIGNAL RenderList::appendedObject( RobotModel::DisplayList* ) ---> SLOT GLWidget::addDisplayList(DisplayList*) SIGNAL RenderList::outdatedDisplayList( int idx ) ---> SLOT GLWidget::removeDisplayList(int) SIGNAL RenderList::changedState() ---> SLOT GLWidget::update() SIGNAL GLWidget::renderStuff() ---> SLOT RenderList::callLists()

NOTE: The signal slot mechanism ensures that display lists are created, destroyed and called by the same thread. This is necessary because OpenGL is not thread safe.
