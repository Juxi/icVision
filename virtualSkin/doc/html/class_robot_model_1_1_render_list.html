Detailed Description

An abstract class that provides the signal/slot interface between GLWidget and the (physical) objects to be rendered therein.

This class should be inherited by any class that contain a number of physical objects to be rendered. See Robot and World as examples. See GLWidget for an explanation of how to connect the signals and slots.
