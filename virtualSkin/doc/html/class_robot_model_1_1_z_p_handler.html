Detailed Description

A custom handler for the Qt XML parser.

This code is mostly borrowed from the QtXML example. The functions startElement(...), and endElement(...) have been reimplemented to handle our XML specification, and the function createChildItem() in the QtXML example has become createChildBodyPart(), createChildMotor(), and createChildJoint().TODO: Document XML Spec
