Detailed Description

A simple circular buffer of QVectors.

In VirtualSkin, this is used to store the history of poses for a RobotModel::BodyPart. The history is written to the buffer by StateObserver. The outer QVector contains states of the RobotModel::BodyPart, and the inner QVector (each state) contains a tuple of motor encoder positions.
