Detailed Description

Real matrix, supporting basic linear algebra operations as well as eigen-decomposition, singular value decomposition, and operations building thereon, such as matrix exponential.

NOTE: Eigen-decomposinion, matix square root and exponential assume a SYMMETRIC square matrix. For non-symmetric matrices the result is undefined!
