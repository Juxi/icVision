Detailed Description

Provides an RPC interface (via YARP) to RobotModel::World.

This interface is compatible with that of the iCub simulator in terms of the syntax of RPC commands, the kinds of geometries supported and the way those geometries are named. However in addition to individual primitives, the World supports CompositeObjects, made up of unions of PrimitiveObjects.
