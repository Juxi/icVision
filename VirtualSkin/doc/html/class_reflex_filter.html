Detailed Description

A VirtualSkin::RobotFilter with reflexive collision response.

This Filter forwards commands from the client (a control program) until the internal robot model detects a dangerous configuration. At that time, the client is cut off from the robot, the Filter moves the robot back to a "safe" configuration, and then control is returned to the client program.
